# Cadei-Rotas
Projeto da matéria Projeto Integrador de Fundamentos UnB 2026/1.

# Sobre o Projeto
O Cadei-Rotas é um aplicativo mobile de acessibilidade para cadeirantes que mostra pontos acessíveis (como rampas, elevadores e banheiros) em um mapa da UnB e dá a opção para o usuário (cadeirante ou não) denunciar os pontos não acessíveis (calçadas esburacadas, rampas íngremes e caminhos de difícil acesso).  

O próximo passo é implementar, dentro do mapa, rotas que evitam os pontos inacessíveis e priorizam os pontos acessíveis.
